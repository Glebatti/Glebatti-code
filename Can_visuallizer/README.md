# CAN-лог визуализатор

Простое веб-приложение для загрузки и визуализации CAN-логов.  
Состоит из бэкенда на FastAPI и фронтенда на Flask.

## Возможности
- Загрузка CSV-файла с колонками: timestamp, id, data.
- Просмотр списка доступных идентификаторов (ID).
- Построение графика выбранного ID.

## Технологии
- Python 3.9+
- FastAPI (бэкенд)
- Flask (фронтенд)
- Pandas (обработка данных)
- Chart.js (графики)
- HTML/CSS/JavaScript
