Поиск и траблшутинг(охота на ошибки) в Linux с помощью grep

Grep это утилита командной строки Linux, который даёт пользователям возможность вести поиск строки. С его помощью можно искать конкретные слова в файле. Также можно передать вывод любой команды в grep, что сильно упрощает работу во время поиска и траблшутинга.
Команда grep часто использует как фильтр для других команд, но ее также можно использовать отдельно

Простой поиск
1. Если требуется найти конкретную папку или один файл среди сотни других, то необходимо передать вывод команды ls в grep через вертикальную черту (|), а уже grep-у параметром передать нужное слово.
	ls | grep <имя файла>
Если команда grep ничего не вернула, значит искомого файла/папки не существует в данной директории.
2. Если требуется найти не одно слово, а словосочетание или целое предложение, то параметр команды grep должно быть выделено кавычками. 
Примечание: grep поддерживает как одинарные, так и двойные кавычки.
	ls | grep '<имя искомого файл или папки>'
3. Для поиска указанных слов в кавычках в файле использовать конструкцию:
	grep '<искомое слово в файле>' <файл>
Примечание: можно искать пробелы, написав grep " ".
4. Для поиска по нескольким параметрам - перед каждым параметром используется ключ <-e>. Данный ключ система понимает как 'или - или' и выводит все вхождения указанных слов.
	grep -e '<искомое слово в файле>' -e '<искомое слово в файле>' <файл>
5. Для поиска без учета регистра - добавить ключ <-i> к команде поиска:
	grep -i <искомое слово в файле> <файл>
6. Для вывода всех папок в текущей директории использовать команду:
	ls | grep '/'
7. Найти точное соотвествие искомого слова командой:
	grep '\<<слово>\>' <файл>
Пример: grep '\<apple>\' fruits.txt

Рекурсивный поиск
1. Для того, чтобы вести поиск по указанному шаблону среди всех файлов во всех папках и подпапках - команду grep нужно запустить с ключом r.
Команда выведет все файлы, где найдено совпадение с указанным шаблоном, а также путь к ним. По умолчанию поиск ведется по текущей директории и поддиректориях.
	grep -r <искомое слово в файле> /путь/искомой/директории

Регулярные выражения
1. Для поиска по одному из указанных символов используется команда:
	grep '<искомое слово в файле [один или несколько указанных символов]>' <файл>
2. Знак дефиса означает диапазон значений:
	grep '<искомое слово в файле [начальный диапазон - конечный диапазон]>' <файл>
3. Знак ^ означает, что команда выведет все строки, которые начинаются с нужного символа:
	grep '<^искомый символ>' <файл>
Примечание: ключ ^ между квадратными скобками исключает из поиска следущие за ним символы или диапазон символов.
4. Знак $ в конце строки выведет только те строки, в конце которых встречается укаданный символ:
	grep '<искомое слово или символ$>' <имя файла>
5. Знак . означает один любой символ из введенных:
	grep '<искомое слово или символ . искомое слово или символ>' <имя файла>

Исключить шаблон
1. Посмотреть содержание файла с исключением конкретной строки к запросу требуется добавить ключ <-v>:
	grep -v '<искомое слово или символ>' <имя файла>
